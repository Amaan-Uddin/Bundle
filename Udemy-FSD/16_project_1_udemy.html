Project HTML 1:

Syntax:

The syntax for our html is quite simple, it consists of an opening tag and a closing tag.
For Ex. "<element>" and "</element>".
The text between our element tag will be executed according to the properties of our tag.
 

Base Structure:

The Base Structure of our html project consists of <!DOCTYPE html>, this informs the browser that the file being executed/runned is a html file.
 
Next up we use the the <html> tag so we could give it a proper foundation.
The <html> tag consist of <head> and <body> tags/elements
We could give a title to our webpage inside of our <head> tag and at the same time we could use it to style our <body> using CSS.
The <body> is where the real html coding takes place. The MAIN part of any html programs.
 

Nesting in HTML5:

Nesting in HTML is not as similer to what we learne in python, although it is a code within a code.

The actual terminology is taken as such:

  • When talking about an element , we consider that element as the <parent> element
  • What ever is given inside of the specific "parent" element is now considered as the <child> of that <parent> element.
  • When dealing with nested elements you should make sure all the code is under proper indentation.
  • often times we use the <div> element as it has no special property/meaning and is best suited for nesting problems.
keep in mind that <div> is block element and it takes up all the white spaces which are left in that specific block.
 

Block and Inline Elements:

As we have discused in the previous section <div> is an great example of a block-element.
A Block element takes up extra space i.e. White Space. If you want to check it out for yourself..
  • Go to any webpage and Right-click
  • Press the Inspect option and you can now view the Source Code of that webpage.
  • Hover over a block on that webpage and you can see it being highlighted.
  • Once highlighted you can see that the Block element takes up even the white spaces.
 
Next up we got our Inline element.
An Inline-element takes up only the specific text present inside of it. If you want to check it out for yourself...
  • Go to any webpage and Right-click
  • Press the Inspect option and you can now view the Source Code of that webpage.
  • Hover over a Inline tag on that webpage and you can see it being highlighted.
  • Once highlighted you can see that the Inline element only takes up the specific text present inside of it.
 

Link:

When you browse through different webpages you might come across a link which looks like This
These are called Link(s) and upon interacting with it you'll be lead to another webpage or even another website.
 
To create a link is simple, all you need is the <a> and </a> tags.
The "a" stands for "Anchor" or "hyper-text-anchor", the most frequently used attributes with it are "href" and "target"
 
href stands for "hyper-text-reference" and is where we write down the actual link or path to it.
target is the other frequently used attribute of our <a> tag.
The purpose of the target attribute is it can open our "href" link in another tab.
 

Image Insertion:

We use images in our webpage to make it more presentable and beautiful.
Inserting an image in your webpage is quite simple than it looks, all you need to make sure is that the image is present in your folder.
 
To Insert an image or to add an image to your webpage you will need to use the <img> tag , and the most important attribute used is "src".
src stands for "source" , and you need to specify the relative or complete path of your image in here.
 
Other useful attributes are the "width","height","alt".
The alt attribute will display a message if the image file is corrupt,notfound or if it doesn't exist.
 
  • You can even link an image to another webpage by nesting the <img> tag/element inside of the <a> tag.
  • Make sure that all the attribute values are given in string format.
