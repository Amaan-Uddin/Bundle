CSS 101:

In this section let's discuss about our new chapter CSS 101.

the Syntax for CSS is quite simple , the key elements are selector,property and value.

there are 3 ways for writing CSS:
  • inline, using the style attribute in an HTML element
  • internal, using the <style> element
  • exporting, through importing a .CSS file.

syntax:
Selector { Property : Value; }

Another way to apply CSS would be through the use of Element Selector or by specifing the Element.
syntax:
<style>
 div { border : 1px solid black; }
</style>

Let's head to our next way of applying CSS and that is through Classes
syntax:
<style>
 .anyClass { border : 1px solid black; }
</style>

<div class="anyClass">
  Hello Wolrd!
</div>

Remember : you can also specify multiple classes in the class attribute.
keep in mind that you can also specify the element AND it's Class to Target a specific Element.


Example : h1.anyClass , div.anyClass etc.

We can also apply CSS with the use of the id attribute.
syntax:
<style>
 #anyID { border : 1px solid black; }
</style>

<div id="anyClass">
  Hello Wolrd!
</div>

It works similer to our Class selectors but the only difference here is that it should be unique to each element.

Remember : you should only have 1 id for each individual element.

let's discuss about applying CSS to nested elements.
Example code:
<div>
  <p>
    <i>Lorem ipsum dolor sit amet consectetur adipisicing elit</i>. Ratione itaque iste sequi est harum nisi cum possimus.
    Quibusdam ullam nam, facere harum laboriosam ducimus velit inventore nemo numquam, quasi et.
  </p>
  <br>
  <i>
    italics font
  </i>
<div>

Now imagine a scenario where you need to change the color of a specific <i> tag,
You can obviously add a Class for it or an Id but it's too much time consuming.

To prevent this we can specify the Nested element in our CSS style sheet.

<style>
   div p i {
      color : blue;  
   }
</style>

by doing this you're targeting a specific element to apply your CSS to.
Result:
Lorem ipsum dolor sit amet consectetur adipisicing elit. Ratione itaque iste sequi est harum nisi cum possimus.
Quibusdam ullam nam, facere harum laboriosam ducimus velit inventore nemo numquam, quasi et.

italics font

as you can see it did not change the color of the other <i> element.

Remember : To specify the order of the nested element correctly.

let's dive into some pseudo-selectors and let's apply some CSS with it.

pseudo-selectors work in various ways , let's use some on a link.
<div>
   Lorem ipsum dolor sit amet consectetur adipisicing elit. <a href="https://youtube.com">Dicta</a>, debitis!
</div>

Result:
Lorem ipsum dolor sit amet consectetur adipisicing elit. Dicta, debitis!

Now let's try and change the color of the our link and also make it so that it changes when the pointer hovers over it and when we press and hold the link.
<style>
   a {
     color : black;
     font-family : Georgia, 'Times New Roman', Times, serif;
   }

   a:hover {
     color : blue;
   }

   a:active {
     color : grey;
   }
</style>

<div>
   Lorem ipsum dolor sit amet consectetur adipisicing elit. <a href="https://youtube.com">Dicta</a>, debitis!
</div>

Result:
Lorem ipsum dolor sit amet consectetur adipisicing elit. Dicta, debitis!

now when you hover over the link it changes it's color and when selected it changes it to grey.
