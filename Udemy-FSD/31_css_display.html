CSS:101

Display

In this section we'll be learning more about inline and block elements, specifically about how we can overwrite these elements.

As we are aware of our <div> element being a block element, another example would be <h1> elements.
If we were to write some code on the same line a block element , it'll force the program to display the other code on the next line and the block element will take up the entire block.

Example:
<h1>Hello</h1>world.
<span>welcome to the world of front-end engineering.</span> How are you ?

Result:

Hello

world. welcome to the world of front-end engineering. How are you ?

to prevent this from happening we can overite the display property of the a block element.

CSS:
div {
   display : inline;
   border : 1px red;
}

<div>
   This is block 1
</div>

<div>
   This is block 2
</div>

Result:
This is block 1
This is block 2
