In this section we'll be discussing about code formating in html.
As we are aware of the fact that our html is not "white space" sensitive and it doesn't include the additional white spaces between 2 characters.
So to add additional white space we use the <br> or &nbsp;

But there are other ways of doing this and that is through the use of <pre> tag.
This tag IS "white-space" sensitive and will add the additional white space between 2 Characters.

Since it's a differnet tag the font for it also quite different, Example...
